# Quickstart

This quickstart brings up the Governix admin console and API locally.

## Prerequisites

- Node.js 20+
- npm
- Docker Desktop or Docker Engine

## Start locally

1. Install packages.

```bash
npm install
```

2. Copy the local environment file.

```bash
cp .env.example .env
```

3. Start PostgreSQL.

```bash
docker compose up -d
```

4. Run migrations and seed demo data.

```bash
npm run db:migrate
npm run db:seed
```

5. Start the app.

```bash
npm run dev
```

6. Open `http://127.0.0.1:3000/login`.

## Seed credentials

- Admin email: `admin@acme.io`
- Admin password: `governix-admin`
- Seed runtime API key: `govx_demo_app_key`

## Common commands

```bash
npm run dev
npm run typecheck
npm run db:migrate
npm run db:seed
PLAYWRIGHT_BASE_URL=http://127.0.0.1:3000 npm run test:e2e
```

## What the seed data gives you

- Demo tenants
- Demo applications
- Demo tenant policies
- Demo quotas
- Initial ledger and usage data
